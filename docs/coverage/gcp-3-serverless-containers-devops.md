Excluded from scope families (no row — fail the "has a hashicorp/google Terraform resource" test):
- **Container Registry** — no `google_container_registry_*` resource exists in the provider, only a read-only `google_container_registry_repository` data source (GCR is a GCS-bucket convention with nothing to manage).
- **Batch** — no `google_batch_job` (or any `google_batch_*`) resource exists in the provider despite `batch.googleapis.com/Job` being CAI-enumerable.
- **Cloud Run for Anthos** — not a distinct resource; provisioned via `google_cloud_run_service` targeting a GKE-backed Cloud Run add-on (see note on that row).
